Q.4 "Create a promise function accepting an argument, if yes is passed to the function then it should go to resolved state and print Promise Resolved, and if nothing is passed then it should go to reject the state and catch the error and print Promise Rejected " 

        function demo(a){
            return new Promise((res,rej)=>{
                if (a=="yes")
                    res("Promise Resolved")
                else
                    rej("Promise Rejected")
                       })
                
            }
        demo("yes").then((output)=>{
            console.log(output)
        })
        .catch((err)=>{
            console.log(err)
        })

output:Promise Resolved

Explaination:In this program i wrote promise function accepting the argument and promise is in the inside of demo function so we called demo with argument like demo("yes") and inside of promise checking the argument with "yes" then if it is executed will get Promise resolved other wise will get Promise rejected

Q. 7 Create examples to explain promises function 

        let myPromise = new Promise(function(Resolve,Reject) {
            let x = 0;
            if (x == 1) {
              Resolve("i am the promise function");
            } else {
              Reject("error");
            }
          });
          
          myPromise.then(
            function(value) {console.log(value);}
            
          )
          .catch(
              function(error) {console.log(error);})

output:i am the promise function

Explaination:A promise is an object to handle the asynchronous operations and It is used to find out if the asynchronous operation is successfully completed or not. A promise has three kind of states like pending and resolved , rejected. if task was executed then we get result otherwise error

Q. 8 Create examples to explain async await function

        async function demo() {
            let promise = new Promise(function(resolve, reject) {
              resolve("welcome to real world");
            });
            var res=await promise;
            console.log(res)
          }
          
        demo();

output:welcome to real world

Explaination:Normally all assigned asynchronous operations are start at same time but here in my code i used async keyword to the function like async function demo() and there is await keyword in my function here the purpose await keyword is wait for the result of that statement, and completion of the statement then only it will move to next line
